# ANALYSIS.md - Dataset Generation Model Evaluation

## Overview
This document provides a comprehensive analysis of synthetic datasets generated by different Gemma model variants (1B, 4B, 12B, 27B) for Estonian conversational data generation. The evaluation focuses on data quality metrics, class balance, consistency, and overall dataset characteristics.

## Dataset Overview

### Models Evaluated
- **Gemma 3 1B (Quantized)**: gemma3:1b-it-qat - Smallest quantized model variant
- **Gemma 3 4B (Quantized)**: gemma3:4b-it-qat - Mid-size quantized model variant  
- **Gemma 3 12B (Quantized)**: gemma3:12b-it-qat - Larger quantized model variant
- **Gemma 3 27B (Quantized)**: gemma3:27b-it-qat - Largest quantized model variant

**Note**: All models use Instruction-Tuned Quantization (QAT) variants, which provide:
- Reduced memory footprint compared to full-precision models
- Maintained performance through quantization-aware training
- Improved inference speed and deployment efficiency

### Dataset Characteristics
- **Target Language**: Estonian
- **Data Format**: Conversational dialogues
- **Number of Agencies**: 3 (Id.ee, PPA, Tarbijakaitse)
- **Total Topics**: 643 (72 + 487 + 84)
- **Total Conversations**: 3,215 (360 + 2,435 + 420)

#### Agency-Specific Breakdown
- **Id.ee (Identity and Status Agency)**
  - Topics: 72
  - Conversations: 360
  - Average conversations per topic: 5.0

- **PPA (Police and Border Guard Board)**
  - Topics: 487
  - Conversations: 2,435
  - Average conversations per topic: 5.0

- **Tarbijakaitse (Consumer Protection and Technical Regulatory Authority)**
  - Topics: 84
  - Conversations: 420
  - Average conversations per topic: 5.0

## Cross-Model Comparison

### Dataset Scale Analysis
- **Total Dataset Size**: 3,215 conversations across 643 topics
- **Agency Distribution**: 
  - PPA: 75.8% of conversations (2,435/3,215)
  - Id.ee: 11.2% of conversations (360/3,215)

- **Conversation Density**: Consistent 5.0 conversations per topic across all agencies
- **Topic Complexity**: Varies significantly (PPA: 487 topics vs Id.ee: 72 topics)

1. **Estonian Fluency**: 27B-QAT (0.97) > 12B-QAT (0.92) > 4B-QAT (0.76) > 1B-QAT (0.12)

3. **Conversational Naturalness**: 27B (0.875 coherence) = 12B (0.875) > 4B (0.453) > 1B (0.413)

## Data Quality Concerns

### Class Balance Analysis

#### Agency-Level Balance
- **PPA Dominance**: 75.8% of total conversations may create model bias toward law enforcement topics
- **Topic Distribution Imbalance**: 
  - PPA: 487 topics (75.7% of all topics)
  - Id.ee: 72 topics (11.2% of all topics)
  - Tarbijakaitse: 84 topics (13.1% of all topics)

#### Conversation-to-Topic Ratio
- **Consistent Ratio**: All agencies maintain exactly 5.0 conversations per topic
- **Good Internal Balance**: Each topic receives equal conversation coverage
- **Potential Issue**: PPA's large topic count may dilute quality of individual conversations

**Recommendation**: Consider rebalancing dataset to achieve more equal agency representation (e.g., 33.3% each) to prevent model bias toward law enforcement topics.

### Language-Specific Issues
1. **Code-Switching**: A lot of instances of English-Estonian-Finnish mixing in 1B model
2. **Grammatical Accuracy**: Variable across models, with 1B showing more errors
3. **Register Consistency**: Mixed formal/informal language use

### Content Issues
1. **Factual Accuracy**: Generally good across all models
2. **Response Completeness**: 1B model shows truncated responses
3. **Contextual Relevance**: 1B model scores lowest (0.235), but understandable because of the code mixing

## Consistency Across Metrics

### Inter-Model Consistency
- **Agency confusion**: Perfect across larger models (12B, 27B: 0 pairs) vs issues in smaller models (1B: 75, 4B: 67 pairs)
- **Information coverage**: Dramatic scaling (1B: 0.342 → 27B: 0.895)
- **Relevance**: Massive improvement (1B: 0.182 → 27B: 0.875)
- **Grammar quality**: Critical scaling (1B: 0.12 → 27B: 0.97)

### Intra-Model Consistency
- **1B Model**: Extremely poor and inconsistent across all metrics
- **4B Model**: Mixed performance - good consistency but poor relevance
- **12B Model**: Excellent and consistent high-quality responses
- **27B Model**: Outstanding consistency across all quality dimensions


## Recommendations for Dataset Improvement

### Short-term Improvements
1. **Response Filtering**: Remove low-coherence responses
2. **Balance Adjustment**: Ensure equal representation across topics

## Key Findings Summary

1. **Model Size Impact**: Larger models generally produce higher quality Estonian text
2. **Agency Confusion**: All models perform excellently in maintaining agency boundaries (critical given 3-agency dataset)
3. **Language Quality**: Significant variation between models, with 4B showing best balance
4. **Topic Coverage**: Good across all models, with some depth variations
5. **Diversity vs. Quality Trade-off**: Smaller models more diverse but less coherent
6. **Dataset Imbalance**: PPA dominates with 75.8% of conversations, potentially creating model bias
7. **Scale Appropriateness**: 3,215 conversations provide substantial training data across 643 diverse topics
8. **Consistency**: Perfect 5.0 conversations-per-topic ratio ensures internal balance within agencies
